

# Data Engineering Pipeline
Using open source bruin tool for data engineering pipeline

##  Project Overview
An advanced data analytics project focused on querying, manipulating, and analyzing relational datasets to support corporate data-driven decision-making. This repository showcases enterprise-level ETL processes and complex database management.

##  Tech Stack & Tools
- **Database Systems:** Hive, Hadoop, Relational Databases, DuckDB.
- **Languages:** SQL, HiveQL.
- **Environment:** Linux.

##  Analytical Capabilities Demonstrated
- **ETL Pipelines:** Complete extraction, transformation, validation, and loading of large structured datasets.
- **Advanced Querying:** Utilization of table joins, subqueries, complex aggregations, and data filtering.
